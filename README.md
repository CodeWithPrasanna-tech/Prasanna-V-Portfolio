# Prasanna V — Portfolio Website

## Folder Structure

```
prasanna-portfolio/
│
├── index.html              ← Main HTML file
├── style.css               ← All CSS styles
├── script.js               ← JavaScript (theme, voice, nav, modals)
│
└── assets/
    ├── images/
    │   ├── prasanna.jpg             ← YOUR PHOTO (place here, used in hero + about)
    │   ├── iot-office-thumb.jpg     ← Thumbnail for IoT Office Automation video
    │   ├── line-follower-thumb.jpg  ← Thumbnail for Line Follower Robot video
    │   ├── wristband-thumb.jpg      ← Thumbnail for Smart Healthcare Wristband video
    │   └── badge.jpg                ← Mathelets Club badge (experience section)
    │
    ├── videos/
    │   ├── iot-office-automation.mp4       ← IoT Office Automation demo video
    │   ├── line-follower-robot.mp4         ← Line Follower Robot demo video
    │   └── smart-healthcare-wristband.mp4  ← Smart Healthcare Wristband demo video
    │
    ├── certificates/
    │   ├── certificate1.jpg    ← NPTEL: Joy of Computing Using Python
    │   ├── certificate2.jpg    ← AICTE: Green Skills & AI Internship
    │   ├── certificate3.jpg    ← Edunet: Shell Skills4Future Program
    │   └── certificate4.jpg    ← International Conference Certificate
    │
    ├── resume/
    │   └── Prasanna_V_Resume.pdf    ← Your resume PDF (download button)
    │
    └── fonts/
        (empty — Google Fonts loaded via CDN)
```

## How to Use

1. **Add your photo**: Place `prasanna.jpg` in `assets/images/`
2. **Add videos**: Drop your `.mp4` files into `assets/videos/`
3. **Add certificates**: Place your certificate images in `assets/certificates/`
4. **Add badge**: Place `badge.jpg` (Mathelets Club) in `assets/images/`
5. **Add resume**: Place `Prasanna_V_Resume.pdf` in `assets/resume/`
6. Open `index.html` in a browser — done!

## Features

- ✅ Light/Dark mode toggle
- ✅ AI Voiceover (Web Speech API) — reads each section aloud
- ✅ Resume visualization + PDF download
- ✅ Certificate gallery with click-to-expand modals
- ✅ Project video players with domain tags
- ✅ Responsive for mobile/tablet/desktop
- ✅ Smooth scroll + active nav highlighting
- ✅ Contact form
- ✅ Formal typography: IM Fell English + Libre Baskerville + Lato
- ✅ Mixed colour accent palette (navy, terracotta, forest green, gold, teal)
